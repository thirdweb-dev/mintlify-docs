The thirdweb API provides an OpenAI-compatible chat completion interface optimized for blockchain interactions.

You can use it to:

- Query on-chain data
- Analyze transactions
- Fetch token info and prices
- Execute or prepare contract calls and swaps
- Deploy contracts
- Generate images and search the web

and more.

  Try out the blockchain llm or SDK integration live on playground. 
</Card>

## Get Started

Use the API with the HTTP API directly or with any OpenAI-compatible client library.

<RequestExample>

```javascript filename
fetch("https://api.thirdweb.com/ai/chat", {
  method: "POST",
  headers: {
    "x-secret-key": "<your-project-secret-key>",
  },
  body: {
    messages: [
      {
        role: "user",
        content: "Send 0.01 ETH to vitalik.eth",
      },
    ],
    context: {
      chain_ids: [8453],
      from: "0x1234567890123456789012345678901234567890",
    },
    stream: false,
  },
});
```

</RequestExample>
