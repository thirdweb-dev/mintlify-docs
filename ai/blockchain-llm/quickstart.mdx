The thirdweb API provides an OpenAI-compatible chat completion interface optimized for blockchain interactions.

You can use it to:

- Query on-chain data
- Analyze transactions
- Fetch token info and prices
- Execute or prepare contract calls and swaps
- Deploy contracts
- Generate images and search the web

and more.

<Card title="New Card" icon="gamepad" href="https://playground.thirdweb.com/ai" cta="View Playground">
  Try out the blockchain llm chat or SDK integration live on playground.
</Card>

## Get Started

Use the API with the HTTP API directly or with any OpenAI-compatible client library.

## View Usage

You can see how many tokens have been used through your integration by logging into thirdweb then **Select Project** \> **AI** \> **Analytics**.
