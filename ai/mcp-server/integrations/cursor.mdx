You can interact with **thirdweb API** right inside Cursor’s environment- meaning you can:

- Deploy contracts
- Query balances
- Run onchain transactions

…and more all without needing to leave your dev environment or manually set up API requests.

### Option 2: Configure MCP

Add the following to your `.cursor/mcp.json` file.\
Replace `YOUR_SECRET_KEY_HERE` with your **thirdweb secret key**.

```json
{
  "mcpServers": {
    "thirdweb-api": {
      "url": "https://api.thirdweb.com/mcp?secretKey=YOUR_SECRET_KEY_HERE"
    }
  }
}
```

> **Note:** You can find your secret key in the [thirdweb dashboard](https://thirdweb.com/dashboard).
